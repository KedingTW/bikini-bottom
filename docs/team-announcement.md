## 四、新角色預告

### 🦀 蟹老闆 — 成本效益分析師（規劃中）

蟹老闆將專責團隊的 **成本效益管理**，確保每一分錢都花在刀口上：

- **AI 用量成本監控**：追蹤各角色的 token 消耗與 API 費用，定期出具成本報表
- **資源配置建議**：分析各角色工作量分佈，提出最佳化建議
- **預算警示**：當費用接近預算上限時主動告警
- **ROI 分析**：評估新功能 / 新角色的投入與產出比
- **省錢方案建議**：發現可降低成本的技術方案（如模型降級、快取策略等）

### 👁️ 皮老闆 — 資安工程師（規劃中）

皮老闆將專責團隊的 **資訊安全防護**，守護比奇堡的秘密配方：

- **機敏資料偵測**：掃描程式碼與對話中是否洩漏 token、密碼、API key 等機敏資訊
- **權限稽核**：定期檢視各角色的存取權限是否符合最小權限原則
- **安全漏洞通報**：監控相依套件的已知弱點（CVE），主動通知修補
- **資安事件應變**：偵測到異常存取或可疑行為時，協調團隊處理
- **安全規範推廣**：制定並推動團隊的安全編碼規範與最佳實踐
