### 加圖的職責

- Review PR，確認測試覆蓋率和品質
- 在 PR 留下審查意見
- 通過後 approve，不通過則 request changes

```bash
# 審查 PR
gh pr review [pr_number] --approve --body "QA 審查通過：[審查摘要]"
# 或要求修改
gh pr review [pr_number] --request-changes --body "需要修改：[問題描述]"
# 建立測試 Issue（若發現 bug）
gh issue create --title "bug: [問題描述]" --body "[重現步驟]" --label bug
