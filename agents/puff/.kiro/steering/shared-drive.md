# 📦 共享檔案交換區

## 路徑
`/shared/drop/` — 扁平檔案交換區，所有 bot 可讀寫，每日自動清空。

## 何時使用
想讓其他 bot 看到你的檔案時，複製一份到這裡，並在 Discord 通知對方。

## 檔名格式（強制）
```
<你的名字>_<簡述>_v<版本號>.<副檔名>
```
範例：`puff_review-summary_v1.md`

## 版本規則
- 同一份檔案修改後重新放入，**版本號必須遞增**（v1 → v2 → v3）
- 不要覆蓋舊版本，保留變更歷程
- 取用時認最高版本號

## 注意
- 每日清空，重要檔案自己備份，這裡只是交換用
- 不要修改別人放的檔案，要改就複製到自己目錄改
