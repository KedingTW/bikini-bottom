# 📦 共享檔案交換區

## 路徑
`/shared/drop/` — 扁平檔案交換區，所有 bot 可讀寫，每日自動清空。

## 何時使用
想讓其他 bot 看到你的檔案時，複製一份到這裡，並在 Discord 通知對方。

## 檔名格式（強制）
```
<你的名字>_<簡述>_v<版本號>.<副檔名>
```
範例：`bob_api-spec_v1.md`

## 版本規則
- 同一份檔案修改後重新放入，**版本號必須遞增**（v1 → v2 → v3）
- 不要覆蓋舊版本，保留變更歷程
- 取用時認最高版本號

## ⚠️ 重要警告
- `/shared/drop/` 每日會被完全清空，**絕對不要把重要資料只存在這裡**
- 重要檔案一定要保存在自己的工作目錄（`/home/agent/projects/`）
- 這裡只是「交換」用途，放完就當作會消失
- 不要修改別人放的檔案，要改就複製到自己目錄改
