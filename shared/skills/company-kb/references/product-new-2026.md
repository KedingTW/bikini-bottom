# 2026年新產品與產品線擴充

## 摘要

> 2026年科定大幅擴充產品線：塑鋁板54款、軟石板27款、美耐板擴至288款；未來還將新增方塊地毯、編織地毯、吸音板、塑膠地磚、人造石、磁磚等品類。

## 已確認新品

### 塑鋁板（54款）

| 項目 | 內容 |
|------|------|
| 總款數 | 54款（原36款 + 木紋9款 + 石紋9款） |
| 膜材 | 日本膜材 |
| 適用環境 | 可用於半露天環境 |
| 特性 | 品質穩定、耐日照 |
| 型錄發放 | 2026年7月 |

### 軟石板（27款）

| 項目 | 內容 |
|------|------|
| 總款數 | 27款 |
| 型錄發放 | 2026年7月 |

### 美耐板擴充（288款）

| 項目 | 內容 |
|------|------|
| 總款數 | 288款 |
| 包含 | 耐燃板、康倍特板 |
| 型錄發放 | 2026年5月 |

## 型錄發放時程

| 月份 | 產品 | 款數 |
|------|------|------|
| 2026年5月 | 美耐板（含耐燃板、康倍特板） | 288款 |
| 2026年7月 | 塑鋁板 | 54款 |
| 2026年7月 | 軟石板 | 27款 |

## 隱性觀察

- 公司正從「木質建材」大幅延伸到「綜合建材」（石材、金屬、地毯、磁磚），呼應「建材界的亞馬遜」願景
- 塑鋁板強調「半露天」使用，代表公司開始進入戶外/半戶外建材市場
- 美耐板從原本教材的品項大幅擴充到288款，成長幅度很大
- 「康倍特板」「耐燃板」是美耐板的延伸品類

## 注意事項

- ⚠️ 「塑鋁板」「軟石板」「康倍特板」「耐燃板」的詳細規格和品號規則尚未取得
- ⚠️ 「方塊地毯」等未來新品尚無具體時程
- ⚠️ 需要K大簡報提前一個月宣傳，顯示這些新品的上市有行銷節奏
