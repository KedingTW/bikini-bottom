# 塗裝木皮板（K6）產品詳細

## 摘要

> K6塗裝木皮板是科定起家產品，結構為「KD獨家UV漆+實木皮+夾板」，分天然木與科技木兩系列，主打工廠預塗裝縮短80%噴漆工期。

## 結構

KD獨家UV漆 + 實木皮 + 夾板

## 系列

| 系列 | 說明 |
|------|------|
| 天然木 | 由原木直接切成薄片木皮，保留天然木紋 |
| 科技木 | 又稱人造木，將木材壓製成木方後再切成木皮，紋理較整齊 |

## 產品特色

1. 木皮因生長環境差異，紋理各具獨特性
2. 專利KD UV漆完整包覆木皮，不易受潮、抗污耐用
3. 工廠預塗裝，縮短80%（4/5）噴漆工期
4. 現場免噴漆、免砂光，減少粉塵及有害氣體污染
5. 翻花拼接使木紋連續延伸，兩片板16尺，最多四片板32尺
6. 可彎板可應用於曲面設計

## 表面處理方式

| 處理方式 | 說明 | 品號特徵 |
|----------|------|----------|
| 毛細孔 | UV塗裝技術，完整保留木皮天然凹凸紋理 | K6 1XX（100~199） |
| 鋼刷 | 鋼刷刷過木皮表面，使木紋凹凸更明顯 | K6 1XX（100~199） |
| 鋸痕 | 在木皮表面留下鋸切紋路，粗獷自然效果 | K6 4XX（400~499） |
| 3D | 表面具備立體凹凸效果，提升木紋層次與觸感 | - |

## 拼法

| 拼法 | 說明 | 品號尾碼 |
|------|------|----------|
| 自然拼 | 直紋木皮拼接 | N 或 U |
| 實木拼 | 山紋木皮與直紋木皮交錯拼接 | S 或 E |
| 水波紋自然拼 | 帶有水波效果的直紋木皮拼接 | U |
| 水波紋實木拼 | 帶有水波效果的直紋+山紋交錯拼接 | E |
| 木節實木拼 | 帶有木節效果的直紋+山紋交錯拼接 | - |

## 顏色分類

| 類型 | 說明 |
|------|------|
| 本色 | 未上色，呈現木材原本顏色，不同批次可能略有深淺差異 |
| 塗裝做色 | 透過塗裝調整顏色，色彩更均勻 |
| 水染木皮 | 以染料將木皮染色，顏色滲入木材，色澤較清透 |

## 名詞解釋

- **翻花**：木皮板短邊上下翻轉的方式拼接，形成連續又有變化的紋理效果
