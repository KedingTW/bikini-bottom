# 環保批批板（P）產品詳細

## 摘要

> PP板是科定目前銷售第一的產品，結構為「防刮耐磨塗層+PP超擬真印刷膜+夾板」，以食品級PP原料搭配日本超擬真印刷技術，款式涵蓋木紋、石紋、金屬紋等。

## 結構

防刮耐磨塗層 + PP超擬真印刷膜 + 夾板

## 產品特色

1. 食品級PP（聚丙烯）原料，健康無毒又環保
2. 日本超擬真印刷技術，掃描自天然材料，搭配木紋壓紋增加擬真度
3. 木紋顏色可對應塗裝板
4. 款式豐富：純色、木紋、石紋、異材質等花色
5. 可搭配同色封邊條、同色蠟筆及角線漆料
6. 多片板材拼接紋理可延續

## 品號編碼規則

品號格式：`P` + 木種/類型編碼 + 色號

### 木紋類編碼區間

| 編碼區間 | 紋理類型 | 說明 |
|----------|----------|------|
| 1XX | 綜合紋理（直紋為主） | 含直紋、水波、白毛孔 |
| 2XX | 山紋 | Crown Cut |
| 3XX | 膚感/立體紋 | Silky Matte |
| 4XX | 鋸痕 | Rough Cut |
| 5XX | 橫貼 | - |
| 6XX | 立體山紋/泥紋 | - |

### 異材質/純色編碼區間

| 編碼區間 | 類型 |
|----------|------|
| 70~79 | 異材質（石紋、清水模、金屬紋等） |
| 80~86 | 純色 |
| 87~99 | 異材質 |

## 板厚度

- 標準厚度：3mm

## 注意事項

- ⚠️ PP板（P開頭）為臺灣/大陸市場使用的品號前綴
- ⚠️ 木紋顏色「可對應塗裝板」意味著設計師可以在同一空間混搭K6和P板，維持色彩一致性
