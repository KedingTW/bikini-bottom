# 業務職能代號對照表（業一～業八）

## 摘要

> 公司內部溝通統一使用「業一～業八」代號代表不同業務職能：經營/開發/外勤/門市/物流/電訪/產品顧問/銷管。

## 代號對照

| 代號 | 職能 |
|------|------|
| 業一 | 經營 |
| 業二 | 開發 |
| 業三 | 外勤 |
| 業四 | 門市 |
| 業五 | 物流 |
| 業六 | 電訪 |
| 業七 | 產品顧問 |
| 業八 | 銷管 |

## 已知資訊補充

- **業二（開發）**：負責新客開發，2026年4月會議提到「業二開發效益佳，主因為新品效益及系統性開發」
- **業一（經營）vs 業三（外勤）**：內外部職稱對照表已公告，「對外業一可與業三同階」— 意即經營人員對外可使用與外勤同等級的職稱
- **業六（電訪）**：負責打電話邀約客戶參加K大；資深業六可兼任業七工作
- **業七（產品顧問）**：負責K大的產品解說（含Q&A真人回答）
- **業六→業七**：是一條職涯成長路徑，業六資深後可勝任業七

## 隱性觀察

- 這不是傳統的「業務一部/二部/三部」按區域分，而是按**職能**分的代號系統
- 涵蓋範圍很廣：從前端（開發/經營/外勤/門市）到後端（物流/銷管）都納入「業」的體系
- 「電訪」獨立為一個職能（業六），顯示電話行銷/客服是重要的業務管道
- 「產品顧問」（業七）獨立存在，可能是技術型業務角色
- 「對外業一可與業三同階」：經營（業一）對外可用外勤（業三）的職稱，暗示外勤的對外職稱較高

## 注意事項

- ⚠️ 各代號的具體職責範圍、人數規模待補充
- ⚠️ 「對外業一可與業三同階」的具體職稱名稱待確認
