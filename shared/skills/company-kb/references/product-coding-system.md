# 產品品號編碼規則

## 摘要

> 科定品號由「產品前綴+木種編碼+色號+拼法代碼」組成，不同產品線共用木種編碼邏輯，但各有特殊規則。

## 品號結構總覽

### 塗裝板（K6）品號結構

```
K6 [木種編碼3碼] [色號1碼] [拼法1碼]
例：K6 187 B S
    │   │    │  └─ S=實木拼, N=自然拼, U=水波自然拼, E=水波實木拼
    │   │    └──── 色號（A/B/C...代表不同顏色）
    │   └───────── 木種編碼（187=白橡木）
    └───────────── 產品前綴
```

### 批批板（P）品號結構

```
P [木種/類型編碼3~4碼] [色號1碼]
例：P 287 B
    │  │   └─ 色號
    │  └───── 木種編碼（287=白橡木山紋）
    └──────── 產品前綴
```

### 木地板（K3）品號結構

```
K3 [木種編碼3碼] [色號1碼]
例：K3 187 B
    │   │    └─ 色號
    │   └────── 木種編碼（187=白橡木）
    └────────── 產品前綴
```

## 拼法代碼（塗裝板 K6）

| 代碼 | 拼法 | 英文 |
|------|------|------|
| S | 實木拼 | Mismatch |
| E | 實木拼（水波） | Mismatch (Figure) |
| N | 自然拼 | Quarter Mismatch |
| U | 水波紋自然拼 | Quarter Mismatch (Figure) |

### 重要：同木種不同編碼

| 木種 | 編碼A | 編碼B | 差異 |
|------|-------|-------|------|
| 柚木 | 118 | 186 | 118=毛細孔處理, 186=鋼刷處理 |
| 胡桃木 | 121 | 181 | 121=毛細孔處理, 181=鋼刷處理 |

## 塗裝板特殊編碼區間

| 區間 | 表面處理 |
|------|----------|
| 100~199 | 天然木種（毛細孔/鋼刷） |
| 400~499 | 天然木種（鋸痕） |

## 批批板編碼區間

### 木紋類

| 區間 | 紋理類型 |
|------|----------|
| 1XX | 綜合紋理（直紋為主） |
| 2XX | 山紋 |
| 3XX | 膚感/立體紋 |
| 4XX | 鋸痕 |
| 5XX | 橫貼 |
| 6XX | 立體山紋/泥紋 |

### 異材質/純色類

| 區間 | 類型 |
|------|------|
| 70~79 | 異材質（石紋、清水模、金屬紋等） |
| 80~86 | 純色 |
| 87~99 | 異材質 |

## 注意事項

- ⚠️ 批批板（P）和美耐板（L）的數字編碼邏輯完全相同，僅前綴不同
- ⚠️ K6的478=白栓木鋸痕，是400~499區間的例子
- ⚠️ 色號字母不代表固定顏色，同一木種的不同色號代表不同花色/深淺
