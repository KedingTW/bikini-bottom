# 科定企業集團概覽

## 摘要

> 科定企業（KD）為臺灣上市建材公司，2002年創立，以塗裝板起家，目標成為「建材界的亞馬遜」，全球佈局涵蓋亞太、歐美、中東、非洲。

## 基本資訊

| 項目 | 內容 |
|------|------|
| 公司全名 | 科定企業股份有限公司 |
| 品牌名稱 | KD |
| 創辦人/董事長 | 曹憲章 |
| 創立年份 | 2002年 |
| 上市時間 | 2018年（臺灣證券交易所） |
| 員工數 | 約800人（截至2026年2月） |
| 市值 | 約100億（截至2026年2月） |
| 深耕年數 | 24年（截至2026年） |

## 發展里程碑

| 年份 | 事件 |
|------|------|
| 2002 | 創立，以「科定塗裝板」奠定業界領導地位 |
| 2017 | 建置嘉義生產總部 |
| 2018 | 臺灣證券交易所掛牌上市 |
| 2022 | 定調「AI生態圈」策略 |
| 2024 | 台北企業總部落成 |
| 2025 | AI生態圈全面深化運用，取得階段性成果 |
| 2026 | 新樹物流園區啟動 |

## 核心戰略與願景

- **終極目標**：成為「建材界的亞馬遜」
- **品牌定位**：
  - 深化客戶腦海中「KD綜合建材廠商」的專業形象
  - 確立消費者心中「KD綠建材廠商」的領先地位
- **AI戰略**：2022年定調「AI生態圈」策略，至2025年已全面深化運用
