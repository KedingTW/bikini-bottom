# 科定最新動態與會議決策

## 適用場景

當需要了解公司最新決策、政策變動、近期會議重點時啟用此 skill。

## 2026年4-5月午餐會議重點

### 2026/4/6~4/10
- 海外開發策略：現階段以**薄板**為主力產品
- **塑鋁板**擴充：原36款 → 共54款（+木紋9款+石紋9款）
- 塑鋁板特性：日本膜材，品質穩定，可用於半露天環境，耐日照
- K大簡報需提前一個月宣傳新品

### 2026/4/13~4/30
- 業二開發效益佳（新品效益+系統性開發 → 提升新客業績）
- 年度感恩禮併中秋節發送（暫定9月初），親送為主
- 內外部職稱對照表已公告，對外業一可與業三同階
- 集團落實每年60小時教育訓練
- **印尼據點自4/1起從直營轉為經銷**
- 廣交展具加盟開發潛力；同步評估新加坡展會
- 台灣展示館營業時間調整（統一9:00-18:00）

### 2026/5/4~5/8
- 未來週一至週日皆安排值班人員接單+K大宣導
- 週一至週六全台皆有送貨
- **澳洲、美國市場業務據點設於菲律賓**
- 海外新增產品：星光板、超耐磨、SPC、木門組；建商開發
- 未來新增產品：方塊地毯、編織地毯、吸音板、塑膠地磚、人造石、磁磚
- 型錄發放：5月美耐板288款 / 7月塑鋁板54款+軟石板27款
- **6月漲價**：透芯美耐板、PP板、PP木門組

### 2026/5/11~5/15
- 耐燃板命名確認：**火盾板**
- 未來將引進**人造石、石英石**產品銷售
- 名片改版：背面列入科定全品項資訊
