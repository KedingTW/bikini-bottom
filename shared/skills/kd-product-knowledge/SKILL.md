## 飾面材共同特色

1. 低甲醛釋放，達F1等級
2. 表面防刮耐磨塗層
3. 耐酸鹼、耐髒污，可用酒精擦拭
4. 耐極端溫差
5. 不易受潮、變色
6. 不含甲苯、八大重金屬
7. 防焰一級（遇火不易燃燒，離火自熄）
8. 夾板底材可替換矽酸鈣板，可達耐燃一級

## 塗裝木皮板（K6）

- **結構**：KD獨家UV漆 + 實木皮 + 夾板
- **系列**：天然木 / 科技木
- **核心賣點**：工廠預塗裝，縮短80%噴漆工期；現場免噴漆免砂光
- **表面處理**：毛細孔(100~199) / 鋼刷(100~199) / 鋸痕(400~499) / 3D
- **拼法**：自然拼(N/U) / 實木拼(S/E)
- **可彎板**：可應用於曲面設計

## 環保批批板（P）

- **結構**：防刮耐磨塗層 + PP超擬真印刷膜 + 夾板
- **厚度**：3mm
- **核心賣點**：食品級PP原料；日本超擬真印刷；款式豐富（木紋/石紋/金屬紋/純色）
- **木紋顏色可對應塗裝板**（同空間可混搭K6和P板）

## 環保美耐板（L）

- **結構**：防刮耐磨塗層 + PP超擬真印刷膜 + 同色透心層
- **厚度**：1mm
- **核心賣點**：全品項透芯設計；可彎成弧形（最小直徑10cm）；美工刀可修邊
- **主要市場**：海外（SG/PH/TH/IN/VN/BD）

## 星光擬真板

- **結構**：防刮耐磨塗層 + PVC/美耐皿超擬真印刷膜 + 夾板
- **PVC系列(C)**：防水防潮材質
- **美耐皿系列(N)**：熱固性塑膠，不膨脹翹曲；木紋同步壓紋技術
- **定位**：親民價格的擬真飾面材

## 科定塑合板（S）

- **結構**：裝飾印刷層 + 塑合板
- **核心賣點**：木紋可對應塗裝板；同步壓紋技術；抗彎強度佳、耐重不易變形

## 手刮木地板（K3）

- **結構**：KD獨家UV漆 + 實木層(3mm) + 夾板
- **尺寸**：14×180×1800mm
- **核心賣點**：海島型；厚達3mm天然實木皮；手刮立體波紋

## 超耐磨木地板（M）

- **結構**：抗UV超耐磨層 + 擬真印刷層 + HDF高密度密集板 + IXPE防水靜音墊
- **尺寸**：12×196×1835mm
- **核心賣點**：木紋同步壓紋；卡扣免膠；IXPE靜音墊；防潮；商用耐磨等級

## SPC地板（R）

- **結構**：抗UV層 + 耐磨層 + 裝飾紙層 + 基材層 + 礦物基材芯層 + IXPE防水靜音墊
- **尺寸**：木紋 7.5×182×1500mm / 石紋 7.5×457×800mm
- **核心賣點**：防水防潮（高濕度可用）；卡扣免膠；商用耐磨等級

## 連工帶料

### 系統櫃
- 可搭配面材：K6 / P / 星光板(PVC/美耐皿) / 塑合板(S)
- 客製化尺寸，板材防潮抗濕

### 房間門
- 結構：面材 + 底材夾板 + 日本隔音毯(選配) + 全實木門芯 + 高密度夾板塗底框
- 可搭配面材：K6 / P / 星光板(僅PVC)
- 工廠預製，現場快速安裝

## 綜合建材

- **強化木芯板**：香杉木種，天然防蟲，高密度全實木芯材
- **科定強力膠**：塗膠/噴膠，低VOC，瞬間貼合力強
