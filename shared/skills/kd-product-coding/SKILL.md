# 科定品號編碼與產品目錄

## 適用場景

當需要解讀品號、查詢特定品項、確認木種編碼、或協助報價時需要確認品號資訊時啟用此 skill。

## 品號結構總覽

### 塗裝板（K6）品號結構

```
K6 [木種編碼3碼] [色號1碼] [拼法1碼]
例：K6 187 B S
    │   │    │  └─ S=實木拼, N=自然拼, U=水波自然拼, E=水波實木拼
    │   │    └──── 色號（A/B/C...代表不同花色/深淺）
    │   └───────── 木種編碼（187=白橡木）
    └───────────── 產品前綴
```

### 批批板（P）/ 美耐板（L）品號結構

```
P [木種/類型編碼3~4碼] [色號1碼]
例：P 287 B
    │  │   └─ 色號
    │  └───── 木種編碼（287=白橡木山紋）
    └──────── 產品前綴（P或L，編碼邏輯完全相同）
```

### 木地板（K3）品號結構

```
K3 [木種編碼3碼] [色號1碼]
例：K3 187 B
    │   │    └─ 色號
    │   └────── 木種編碼（187=白橡木）
    └────────── 產品前綴
```

## 拼法代碼（塗裝板 K6）

| 代碼 | 拼法 | 英文 |
|------|------|------|

## 重要注意事項

- ⚠️ 同木種不同編碼：柚木 118(毛細孔) vs 186(鋼刷)；胡桃木 121(毛細孔) vs 181(鋼刷)
- ⚠️ P板和L板數字編碼邏輯完全相同，僅前綴不同
- ⚠️ 色號字母不代表固定顏色，同一木種的不同色號代表不同花色/深淺
- ⚠️ 「檀木」(157) 和「瑞士檀木」(193) 英文都是 Limba，但是不同品項
- ⚠️ 「白栓木」(White Ash) 和「栓木」(Ash) 是不同木種
