# 科定報價助手

## 適用場景

當需要協助進行產品報價、查詢價格、建立CRM報價單時啟用此 skill。

## 報價流程總覽

### 標準報價步驟

1. **確認客戶** — 使用 `SearchCustomer` 查詢客戶資料
2. **確認聯絡人** — 使用 `SearchCustomerContact` 取得聯絡人
3. **確認產品與規格** — 使用 `SearchProductItem` 確認品號存在
4. **查詢價格** — 依產品類型使用對應的報價工具
5. **建立報價單** — 使用 `CreateQuote` 在CRM建立記錄

### 報價工具對照

| 產品類型 | 報價工具 | 備註 |
|----------|----------|------|
| 房間門（門組） | `GetRoomDoorPrice` | product_type=door_set |
| 房間門（門片） | `GetRoomDoorPrice` | product_type=door |
| 房間門門框 | `GetRoomDoorFramePrice` | 獨立查詢 |
| 房間門配件/五金 | `GetRoomDoorAccessories` | 分類查詢 |
| 系統櫃門片 | `GetCabinetDoorPrice` | 以才計價 |
| 系統櫃運費 | `GetCabinetShippingFee` | 依地區計算 |
| 木地板 | `GetFlooringPrice` | 依品號+施工方式 |
| 木地板配件 | `GetFlooringAccessories` | 分類查詢 |

## 房間門報價要點

### 材質與系列

| 材質代碼 | 材質名稱 | 系列選項 |
|----------|----------|----------|
| PP | 環保批批板 | A1~F |
| PVC | 星光擬真板 | A1~B |
| K6 | 塗裝板 | natural/artificial |

### 預設參數
- 門洞寬度：90cm
- 門洞高度：220cm
- 牆厚：8.5~16cm（超出自動加價）
- 連工帶料：true（含免運費）
- 有電梯：true

### 門框類型
- pp：批批框（塗裝可搭配）
- pvc：星光擬真框
- skin：膚感框
- primer：塗底框

## 系統櫃門片報價要點

### 計價規則
- 才數 = 長(mm) × 寬(mm) ÷ 90000，四捨五入到小數一位，不足最低才數以最低才數計
- 總價 = round(計價才數 × 才單價 × 數量)
- 台扣門另加基本費500元/片

### 門片類型
背板 / 桶身 / 門片 / 台扣門 / 造型門 / 結烤門(六面) / 霧烤門(六面) / 檯面 / 發泡板

### 產品類別
- P = 環保批批板
- S = 科定塑合板
- N = 星光擬真板(美耐皿)
- C = 星光擬真板(PVC/金屬)

## 木地板報價要點

### 產品線識別（依品號前綴）
- K3 = 手刮木地板
- F = PP環保木地板
- M = 超耐磨木地板
- R = SPC地板

### 施工方式
- direct_lay = 直鋪
- flat_lay = 平鋪
- raised = 架高
- double_flat_lay = 雙層平鋪

### 靜音墊（僅K3/PP適用）
- BZ00083 = 一般靜音墊 $250/坪
- BZ00082 = 降噪靜音墊 $400/坪

## CRM報價單建立

### 必填欄位
- 報價單類型：1=房間門, 2=地板, 3=系統櫃, 4=一般
- 報價單日期：YYYY-MM-DD
- 公司名稱（account_id）：從 SearchCustomer 取得
- 客戶關係連絡人（contact_relation_id）：從 SearchCustomerContact 取得
- 付款方式：1=現金或匯款, 2=月結, 3=訂金30%現金, 4=訂金30%月結, 5=訂金40%現金, 6=訂金40%月結
- 業務申請人

## 價格類型

| 代碼 | 名稱 | 說明 |
|------|------|------|
| designer | 設計師優惠價 | 預設，給設計師/經銷商 |
| msrp | 牌價（業主價） | 給終端消費者 |

## 注意事項

- ⚠️ 2026年6月起透芯美耐板、PP板、PP木門組漲價
- ⚠️ 連工帶料免運費；單純出材料需計算運費
- ⚠️ 牆厚超出8.5~16cm時自動計算加價
- ⚠️ 無電梯時需計算搬運費（依樓層數）
